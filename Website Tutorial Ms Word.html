"The levels of Competency Website Tutorial Ms Word 2013"

Add Another Page?

To put a blank page into your Word document, place the cursor where you want the new page to begin and then click Insert > Blank Page.

Do you know how to create a new document?

In the Search for online templates box, enter a search word like letter, resume, or invoice. Or, select a category under the search box like Business, Personal, or Education.

Do you know how to save files?

Pressing Ctrl + S ( ⌘ Cmd + S on Mac) will save your file in most programs.

Do you know how to rename your files?

Right-click the file and select Rename. Enter a new file name and press Enter.

Do you know how to change fonts and their size?

To change your display in Windows, select Start > Settings > Accessibility > Text size. To make only the text on your screen larger, adjust the slider next to Text size. To make everything larger, including images and apps, select Display , and then choose an option from the drop-down menu next to Scale.

Do you know how to put margins?

Select Layout > Margins. Select Custom Margins. In Margins, use the Up and Down arrows to enter the values you want. Select OK when done.

Do you know how to use paragraph formatting?

Select the paragraph or paragraphs you want to format. On the Home tab, click the Line and Paragraph Spacing command. Click Add Space Before Paragraph or Remove Space After Paragraph from the drop-down menu. ... The paragraph spacing will change in the document.

Do you know how to put bullets and numbering?

On the View tab, click Normal. Click in the text box or placeholder where you want to add bulleted or numbered text. On the Home tab, in the Paragraph group, click Bullets or Numbering. , and begin typing your list. Press Return to create a new list item.

Do you know how to change font styles?

Select Format > Text Styles. ... In the Item to Change list, select All, hen select the font, size, or color you want for all text in the current view. ... Repeat this process for other views.

Do you know how to use text formatting?

Select the text you want to format. To select a single word, double-click it. To select a line of text, click to the left of it. Select an option to change the font, font size, font color, or make the text bold, italic, or underline.

Do you know how to create an outline?

Open Word and click on the “Home” tab. Type out your main headings for each section. Use simple language that lets readers know what each part covers. ... Add subheadings to further break down each section. ... Continue for all main headings and subheadings.

Do you know how to put a photo in a document?

On the Insert tab, click Pictures. Select the option you want to use for inserting pictures. The Photo Browser option helps you browse through existing photo sets on your computer, such as photos in iPhoto or Photo Booth.

Do you know how to input headers & footer?

Go to Insert > Header or Footer. Choose from a list of standard headers or footers, go to the list of Header or Footer options, and select the header or footer that you want.

Do you know how to make a table of contents?

Click where you want to insert the table of contents – usually near the beginning of a document. Click References > Table of Contents and then choose an Automatic Table of Contents style from the list.

Do you know how to ignore ribbon?

Right-click any of the ribbon tabs, and then select Collapse the ribbon. Right-click Ribbon display options in the lower right of the ribbon, and then select Collapse the ribbon. Press CTRL+F1.

Do you know how to use page numbering?

Go to Header & Footer > Page Number > Format Page Numbers, select the format you want, and then select OK. Go to Header & Footer > Page Number > Page Number, choose the alignment you want, and select OK.

Do you know how to use Outline?

Type Your Text: Start by typing the word or phrase you want to outline. ... Select Your Text: Click and drag your mouse to select the text you want to outline. ... Choose the Outline Menu: After clicking the "Text Effects and Typography" tool, a menu will appear.

Do you know how to make a section?

Select where you want a new section to begin. Go to Layout > Breaks, and then choose the type of section break you want. Next Page Starts the new section on the following page. Continuous Starts the new section on the same page.

Do you know how to indent the paragraph?

To indent the first line of a paragraph, put your cursor at the beginning of the paragraph and press the tab key. When you press Enter to start the next paragraph, its first line will be indented.

Do you know how to input shapes?

On the Insert tab, click. Shapes. Select the shape you want, click in the workspace and drag to draw the shape. To create a perfect square or circle (or constrain the dimensions of other shapes), press and hold Shift while you drag.

Do you know how to use Content controls?

In the list of tabs under Customize the Ribbon, select the Developer box and then click OK. Do one of the following: To add a control, click in the document where you want to add it, and then click the control you want to add on the Design tab.

Do you know how to use Version History?

Click at the top of the document where the title of the file appears. In the box that appears, click at the bottom on Version history. You will now see a box on the right with a list of previous versions of a document.

Do you know how to use Document Encryption?

Go to File > Info > Protect Document > Encrypt with Password. Type a password, press OK, type it again and press OK to confirm it. Save the file to make sure the password takes effect.

Do you know how to use a Crop Image?

All you need to do is open your image, select the Crop Tool, drag the corners to mark out the part of the image you want to keep, and hit Enter on your keyboard.

Do you know how to use Equations and formulas?

A formula is a calculation for a specific purpose, for example converting Fahrenheit to Celsius or vice versa. A formula is also always true, no matter what values are put in. An equation will only work for certain values, and is not always true.

Do you know how to make Smart art graphics?

Select Insert > SmartArt. On the left, select the type of layout you want. Select an option and select OK. Select [Text] and type the text you want.

Do you know how to make Cross-references?

On the Insert tab, click Cross-reference. In the Reference type box, click the drop-down list to pick what you want to link to. The list of what's available depends on the type of item (heading, page number, etc.) you're linking to.

Do you know how to use Master documents?

Create new, blank document and save it as "Master" Click View > Outline (in the Document Views section of the tab) ... Click Show Document and then "Insert" ... On the Insert Subdocument dialog box, navigate to the location of the documents you want to insert.

Do you know how to use Macros?

Click View > Macros > Record Macro. Type a name for the macro. To use this macro in any new documents you make, be sure the Store macro in box says All Documents (Normal.dotm). To run your macro when you click a button, click Button.

Do you know how to use a Chart?

Click Insert > Chart. Click the chart type and then double-click the chart you want. ... In the spreadsheet that appears, replace the default data with your own information. ... When you've finished, close the spreadsheet.
